## Quick Start Commands:

### Start both together:
```bash
npm run dev
```

### Start separately:

**Terminal 1 - Backend:**
```bash
cd server
node server.js
# or
npm start
```

**Terminal 2 - Frontend:**
```bash
cd client
PORT=3001 npm start
# or on Windows:
$env:PORT=3001; npm start
```
